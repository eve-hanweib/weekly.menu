# 一周点菜

按周选择早中晚菜品，自动生成菜单和按人数计算的采购清单。

## 功能

- **选菜**：为周一至周日、早/中/晚选择菜品（从 12 道固定菜品中选）
- **菜单**：查看本周菜单汇总
- **采购清单**：根据用餐人数，汇总所有选中菜品所需食材及用量

## 使用

用浏览器打开 `index.html` 即可使用（本地文件或任意静态服务器均可）。

**封面**：首次打开会显示封面（背景图 + “Weekly Menu” 毛玻璃标题），点击后进入选菜页。  
- 使用自定义封面：双击运行 `copy-cover.bat`，会自动把封面图复制为 `cover.png`；或手动将图片保存为 `cover.png` 放在项目根目录。  
- 使用自定义页面背景：双击运行 `copy-bg.bat`，会自动把背景图复制为 `bg.png`；或手动将图片保存为 `bg.png` 放在项目根目录。

## 可选菜品

番茄炒蛋、排骨烧土豆、酱牛肉、三明治、西葫芦虾仁蒸饺、白菜猪肉粉条大包子、清炒上海青、清炒空心菜、大酥牛肉、地三鲜、红烧鱼、炒土豆丝。

## 技术说明

- 纯前端：HTML + CSS + JavaScript，无后端
- 菜品图片使用 Unsplash 免费图库
- 食材用量为“每份/每人”估算，可在 `data.js` 中按需调整
